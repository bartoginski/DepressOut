## Start projektu

Sklonuj projekt

```bash
  git clone git@github.com:bartoginski/DepressOut.git
```

Przejdź do folderu projektu

```bash
  cd DepressOut
```

Zainstaluj zależności

```bash
  npm install
```

Włącz serwer

```bash
  npm run dev
```

Po chwili wpisz w przeglądarce [http://localhost:3000](http://localhost:3000)
